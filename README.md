# www-workymcnet

Source and config for https://workymcnet.work.

## Setup

1. Create a `.env` file
2. [Create a Cloudflare tunnel](https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/get-started/create-remote-tunnel/) and save the token to `.env` as `TUNNEL_TOKEN`.
3. Run `docker compose up -d`.
